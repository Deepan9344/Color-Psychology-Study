# 🎨 Color Psychology Study

> A UI/UX research project that explores how colors influence user emotions, behavior, and decision-making in digital interfaces.

## 📖 Overview

Color psychology is an important aspect of UI/UX design. This project examines how different colors affect user perception and how designers can use colors effectively to create engaging, accessible, and user-friendly interfaces.

## 🎯 Objectives

- Understand the basics of color psychology.
- Study the emotional impact of different colors.
- Conduct a sample user survey.
- Analyze research findings.
- Create practical UI design guidelines.
- Design sample mobile UI screens using color psychology principles.

## 📊 Key Findings

- Blue increases trust and confidence.
- Green creates a feeling of success and positivity.
- Red attracts immediate attention.
- Yellow highlights important information.
- High-contrast color combinations improve readability.
- Consistent color usage enhances user experience.
